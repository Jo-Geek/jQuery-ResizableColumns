Fixed header (This is the default mode, keeps header row fixed)

Example on the right shows a demo of a single row fixed as header.
HTML 
<table class="resizable">
  <thead>
    <tr>
      <th>col 1</th>
      <th>col 2</th>
	  ...
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Column 1</td>
      <td>Column 2</td>
      ...
    </tr>
  </tbody>
</table>
jQuery
$('table.resizable').resizableColumns();
